# 產品背景 (Product Context)

## 問題陳述
目前道騰 (Daoteng) 的服務或內部知識可能分散，亦或是需要一個更智能的方式來回應客戶或員工的查詢。
- 傳統 Wiki 搜尋不易。
- 客戶/員工遇到問題需要人工回覆，效率較低。

## 解決方案
**DT 百科 (DT Smart Client Service App)** - 一個結合 Google Gemini AI 的智能知識庫/客服應用。
- **AI 智能問答**：透過 Gemini API 理解自然語言問題，提供準確解答。
- **知識整合**：集中管理道騰的服務資訊、規章制度或常見問題。
- **即時回應**：提供 24/7 的即時問答服務。

## 目標用戶
- **道騰員工**：快速查詢內部規章、作業流程。
- **道騰客戶**：查詢服務項目、費用、使用規範等 (若為透過客戶端 App)。

## 成功標準
- **回答準確率**：AI 能正確回答大部分常見問題。
- **回應速度**：使用者能即時獲得解答。
- **易用性**：介面直觀，無需學習即可上手。

## 核心功能
1. **AI 對話介面**：類似 ChatGPT 的問答視窗。
2. **知識庫管理**：(預期) 後台管理知識文件。
3. **相關推薦**：AI 根據問題推薦相關文章。
